# yf-digital-pathology-pipeline
Pipeline 100% open-source, auditável e reprodutível para análise de Whole-Slide Images de Febre Amarela.

**Stack:** QuPath 0.7.0 + LazySlide + Slideflow + nnU-Net + UNI2 (Mahmood Lab)
